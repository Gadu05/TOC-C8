## 🎯 The Strategic "Why"

> Navigating a fragmented landscape of academic projects across various repositories and environments can obscure the true breadth of a student's technical capabilities. Demonstrating a cohesive and diverse skill set often requires significant effort in consolidation and presentation, making it challenging for both self-assessment and external evaluation.

TOC-C8 centralizes a student's significant academic works from the "Tópicos de Computadores" course, providing a unified, well-documented portfolio that highlights proficiency across multiple programming languages and frameworks. This repository streamlines skill demonstration and knowledge consolidation, offering a clear and accessible overview of practical application and technical understanding gained throughout the 8th semester of the Computer Science program at IFSP - Campus Salto.

## ✨ Key Features

*   **🌐 Multi-Language Proficiency**: Showcase projects developed in Java, C#, Node.js, and PHP, demonstrating adaptability across various programming paradigms.
*   **🛠️ Diverse Project Types**: Explore a range of applications including APIs, desktop solutions, and mobile components, reflecting comprehensive software development exposure.
*   **📚 Consolidated Learning**: A single point of reference for all major assignments from the "Tópicos de Computadores" course, simplifying review and future reference.
*   **🚀 Practical Application**: Each project demonstrates the hands-on application of theoretical concepts, bridging the gap between classroom knowledge and real-world implementation.
*   **📂 Structured Organization**: Clearly separated project directories ensure easy navigation and understanding of individual assignments and their respective technologies.
*   **📈 Portfolio-Ready**: Provides a tangible and organized portfolio of work, ideal for demonstrating skills to potential employers or for academic review.

## 🚀 Operational Setup

To get these projects up and running on your local machine, follow the steps below. Note that each "Trabalho" (project) is an independent entity and may have its own specific setup instructions within its respective directory.

### Prerequisites

Ensure you have the following runtimes and SDKs installed:

*   **Java Development Kit (JDK)**: Version 11 or higher.
*   **.NET SDK**: For C# projects (e.g., .NET 6.0 or higher).
*   **Node.js**: Version 14 or higher, along with npm or yarn.
*   **PHP**: Version 7.4 or higher, with a web server like Apache or Nginx (and PHP-FPM) for web projects.
*   **Git**: For cloning the repository.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/TOC-C8.git
    cd TOC-C8
    ```

2.  **Navigate to individual projects**:
    Each project (`Trabalho X`) resides in its own directory. You will need to `cd` into the desired project folder to view its specific README and installation instructions.

    *Example for a Node.js project:*
    ```bash
    cd "Trabalho 4 - API Node"
    npm install # or yarn install
    npm start # or yarn start
    ```

    *Example for a Java project:*
    ```bash
    cd "Trabalho 3 - Java"
    # Follow project-specific build instructions (e.g., using Maven or Gradle)
    ```

### Environment Configuration

Some projects, especially those involving APIs or web services, may require environment variables or configuration files (e.g., `.env`, `config.json`). Please refer to the specific README file within each `Trabalho` directory for detailed configuration instructions.
